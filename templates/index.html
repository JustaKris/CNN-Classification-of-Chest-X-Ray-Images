Chest X-Ray Image Classification

This application classifies chest X-ray images using a pre-trained convolutional neural network (CNN). The model can identify whether an X-ray image indicates the presence of three ilnesses - Covid19, Tunerculosis and Pneumonia - with Normal being the output if neither of the three are detected.

How to Use This App

You can upload a chest X-ray image file or provide a URL to the image. The app will display the predicted class and the confidence level. A Grad-CAM heatmap will be generated to highlight the regions of the image that were important for the prediction.
